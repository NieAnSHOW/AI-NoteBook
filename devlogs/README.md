# AI-NoteBook 开发日志

## 📋 命名规范

所有开发日志文件遵循以下命名规则：

```
YYYY-MM-DD-{phase}-{short-description}.md
```

### 命名示例
- `2026-01-20-phase1-init.md` - 项目初始化阶段
- `2026-01-20-phase2-auth.md` - 认证模块开发
- `2026-01-21-phase2-ai.md` - AI 集成开发

### 使用方式
创建新日志时，复制 `TEMPLATE.md` 并重命名：
```bash
cp devlogs/TEMPLATE.md devlogs/$(date +%Y-%m-%d)-{phase}-{desc}.md
```
