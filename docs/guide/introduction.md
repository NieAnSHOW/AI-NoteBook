# 项目介绍

AI-NoteBook 是一个AI驱动的智能内容分析系统，专注于技术文章的深度解析和知识增强。

## 项目背景

在信息爆炸的时代，技术文章和攻略类内容层出不穷，但常常存在以下问题：

- **信息冗余**：大量废话和无关内容影响阅读效率
- **难度不一**：文章难度参差不齐，难以快速评估是否适合自己的水平
- **知识孤岛**：缺乏相关技术背景和应用场景的补充说明
- **信息可信度**：部分技术观点缺乏来源验证

AI-NoteBook 通过AI技术解决这些问题，帮助用户快速获取核心知识。

## 核心功能

### 1. 易读性评分 🎯

对文章进行多维度评分，帮助用户快速判断文章难度：

- **1分**：通俗易懂，适合初学者
- **2分**：需要一定基础
- **3分**：中等难度
- **4分**：需要较深技术背景
- **5分**：专业性强，适合专家阅读

评分维度：
- 词汇难度
- 句式结构复杂度
- 逻辑深度
- 专业术语密度

### 2. 内容提炼 ✨

AI自动过滤冗余内容，保留核心信息：

- 识别文章结构（标题、段落、列表）
- 提取关键论点和结论
- 自动标记重点内容（加粗、下划线）
- 生成结构化大纲

### 3. 智能扩展 🔍

基于重点内容进行知识增强：

- **技术语境**：发展前景、应用场景、关联技术
- **攻略语境**：补充案例、注意事项、最佳实践
- **数据支撑**：实时搜索最新资料，交叉验证

### 4. 溯源校验 🔐

确保信息的可靠性和可追溯性：

- 使用专业搜索引擎获取最新资料
- 多源比对，交叉验证信息
- 标准格式标注引用来源
- 生成可追溯的知识图谱

## 使用场景

### 技术从业者

- 快速筛选技术文章，评估阅读价值
- 获取技术背后的应用场景和发展趋势
- 构建完整的知识体系

### 内容创作者

- 参考优质文章的结构和表达
- 获取相关领域的最新动态
- 验证技术观点的准确性

### 学习者

- 找到适合自己水平的文章
- 获得重点内容的详细解释
- 拓展知识广度和深度

## 产品特色

### 输入方式

- **文本粘贴**：直接复制文章内容
- **文件上传**：支持 .md、.txt 格式

### 输出格式

- **页面预览**：即时查看解析结果
- **导出MD**：保留格式，方便二次编辑
- **导出PDF**：适合打印和分享
- **导出Word**：方便文档协作

## 技术优势

- **前后端分离**：Vue 3 + NestJS，开发体验优秀
- **高性能**：任务队列异步处理，Redis缓存加速
- **可扩展**：微服务架构，易于水平扩展
- **类型安全**：全栈TypeScript，减少运行时错误
- **安全可靠**：JWT + API Key双重认证，数据加密存储

## 快速开始

```bash

git clone https://github.com/yourusername/AI-NoteBook.git

# 安装依赖
cd AI-NoteBook/frontend
npm install

cd ../backend
npm install

# 启动开发环境
npm run dev
```
