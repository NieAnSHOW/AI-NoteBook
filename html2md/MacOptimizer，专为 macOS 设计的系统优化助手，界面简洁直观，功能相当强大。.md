# MacOptimizer，专为 macOS 设计的系统优化助手，界面简洁直观，功能相当强大。

**作者**: Github开源项目
**发布日期**: 2024-09-26

---

今天看到个小玩意儿，有点想立刻安利给所有用 Mac 的人。

就是那个……你用 Mac 用着用着，某天一看「关于本机」，磁盘只剩个几 GB，小风扇呼呼转，你明明啥也没装，空间却莫名其妙就满了，那种窒息感你懂的。

我前段时间就是，被「磁盘空间已满」弹窗轰炸，删照片删视频，甚至把下载文件夹清了个精光，结果发现根本没用——真正占空间的是各种系统缓存、应用残留、一堆不知道从哪冒出来的垃圾文件。

想清理吧，又不敢乱删，生怕一个不小心，把啥系统组件删了，第二天电脑直接起不来。

然后我在 GitHub 上晃悠的时候，偶然刷到一个项目：**MacOptimizer**。看介绍是专门给 macOS 做的系统优化小助手，开源的，界面还挺简洁，不是那种「一键加速、清理 100G 垃圾」的土味软件风格。

我就下下来试了试。

打开之后，它把功能分成了好几个模块：系统监控、应用卸载、垃圾清理、大文件查找之类的，总共八大块，基本把我们日常会用到的那几件事都包了。

我最满意的是 **应用卸载** 这一块。以前删 App 就是拉进废纸篓完事，结果偏好设置在 ~/Library 里躺着，缓存躺着，日志躺着，几百 MB、上 GB 地占着地方。 MacOptimizer 的卸载是那种「深度搜查」模式：一个应用删掉之前，它会把相关的偏好设置、缓存、日志、容器数据、启动项这些统统扫出来，列成清单，你可以自己勾选要留哪个、要删哪个，实在懒得选也可以一键全清。

有点像请了个懂系统结构的朋友，陪你在 Library 里挨个翻，比那种啥都不解释只给你一个「立即清理」按钮的工具踏实多了。

**垃圾清理** 和 **大文件查找** 也挺实用。 垃圾清理会把浏览器缓存、临时文件、旧日志这种东西都集中给你看一眼，你会震惊自己电脑怎么塞了这么多历史遗留； 大文件查找更像是一个放大镜，帮你把几百 MB、几个 GB 的大块头文件揪出来，看看是不是哪部 4K 电影忘在角落里了，或者某次 Xcode 打包留下的构建产物。

对了，它还内置了一个 **文件管理器**，这个我本来以为鸡肋，结果还挺顺手的。 可以直接浏览整个磁盘目录，支持显示隐藏文件，想看 ~/Library 里到底藏了啥，再也不用每次去 Finder 里按快捷键； 而且跟终端还有点小集成，日常你要做一点点小折腾（比如进某个目录跑个命令），也会更方便。

我个人比较在意的一点是：**它没乱七八糟的广告和捆绑**。 很多所谓「清理大师」装上去，自己比垃圾还难卸载，各种常驻、弹窗、订阅提醒，比系统本身还占资源。 MacOptimizer 这种开源项目，代码就摆在那儿，你愿意的话可以自己翻一遍，心理压力会小很多。

安装方式也很友好： 不想折腾的，可以直接下 DMG 装好就用； 习惯命令行的，项目也支持 Homebrew，一行命令装好卸好都干干净净。 Apple Silicon 和 Intel 都适配了，老机器新机器都能用，不挑。

当然，它也不是那种「一键点击，从此世界清净」的魔法工具，更多还是给你一套看得见、可控的工具箱，让你自己决定要动哪里、删什么、留什么。对像我这种既想省事、又有点强迫症、不信任黑盒清理的用户来说，刚好卡在那个舒服的点上。

GitHub 在这儿：http://github.com/ddlmanus/MacOptimizer

有磁盘焦虑症的朋友，可以找个周末下午，备份好数据，慢慢折腾一遍。 说不定清完那一刻，你会突然觉得：嗯，这台老 Mac，还能再战两年。
