# 输出确认模板

每个 Task 结束时的标准化输出确认格式。

## 标准模板

```markdown
#### 输出确认

🔍 **[任务名称]完成**
- 处理项目：{项目说明}
- 执行结果：{结果摘要}
✅ [下一步说明]
```

## 示例

### 需求理解完成

```markdown
🔍 **需求理解完成**
- 核心功能：{功能列表}
- 触发场景：{场景描述}
✅ 可以开始规划内容
```

### 规划完成

```markdown
📋 **规划完成**
- 场景数量：{count}
- 脚本需求：{scripts_list}
- 参考资料：{references_list}
✅ 可以开始实现
```

### 资源实现完成

```markdown
🔧 **资源实现完成**
- 脚本数量：{count}
- 参考文件：{files}
- 测试状态：{passed/failed}
✅ 可以开始编写 SKILL.md
```

### 打包完成

```markdown
📦 **打包完成**
- 文件：{skill-name}.skill
- 位置：{output-path}
✅ 可以分发给用户
```
